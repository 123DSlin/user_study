# GitHub上传步骤

## 方法1：网页上传

1. 访问：https://github.com/123DSlin/0929html
2. 点击 "uploading an existing file"
3. 逐个拖拽以下文件：
   - index.html
   - groupA.html  
   - groupB.html
   - README.md
   - netlify.toml
4. 填写提交信息："Add user study website files"
5. 点击 "Commit changes"

## 方法2：使用GitHub Desktop

1. 下载GitHub Desktop
2. 克隆仓库到本地
3. 复制文件到仓库文件夹
4. 提交并推送

## 方法3：命令行（如果网络正常）

```bash
git add .
git commit -m "Add user study website"
git push origin main
```
